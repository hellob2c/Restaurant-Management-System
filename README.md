# Orders App (Modular)

## 1) Set your Google Apps Script URL
Open `js/config.js` and replace `PASTE_YOUR_EXEC_URL_HERE` with your deployed `/exec` URL.

## 2) Run locally
From this folder:

- Windows:
  py -m http.server 8080
- Mac/Linux:
  python3 -m http.server 8080

Open:
http://localhost:8080/

## 3) Upload to hosting
Upload the entire folder contents (index.html + css/ + js/) to your hosting folder.
